## 📌 Overview

Wheelify is a full-stack web platform that helps Indian vehicle buyers make smarter, data-driven decisions by providing:

- 🔍 **Cross-Category Vehicle Comparison** — Cars, Bikes, EVs, Commercial
- ⛽ **Predictive Fuel Cost Estimation** — Based on real travel patterns
- 📊 **5-Year Total Cost of Ownership (TCO)** — Complete ownership picture
- 💹 **Break-Even Analysis** — EV vs Petrol/Diesel/CNG crossover point
- 🚀 **Upcoming Vehicles Hub** — What's launching in India next
