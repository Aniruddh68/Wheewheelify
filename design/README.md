# Wheelify — Design Assets

All UI/UX design files, exports, and brand guidelines.

## Structure

```
design/
├── stitch-exports/
│   ├── screens/          # Exported HTML/images from Google Stitch (per screen)
│   └── design-system/    # Exported design system tokens
├── assets/
│   ├── mockups/          # High-fidelity mockup images
│   └── wireframes/       # Low-fidelity wireframe images
└── brand/
    ├── colors/           # Color palette reference
    └── typography/       # Font usage guidelines
```

## Brand Colors

| Name       | Hex       | Usage                        |
|------------|-----------|------------------------------|
| Black      | `#0A0A0A` | Primary background           |
| Deep Black | `#050505` | Footer background            |
| Red        | `#FF1A1A` | Primary accent, CTAs, data   |

| Grey       | `#888888` | Secondary text, labels       |

## Typography

- **Display / Headlines**: Space Grotesk — Ultra Bold (700–900)
- **Body / Labels**: Inter — Light to Regular (300–400)
