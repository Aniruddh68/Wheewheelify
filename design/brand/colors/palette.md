# Wheelify Brand Color Palette

## Primary Colors

| Token              | Hex       | RGB                  | Usage                       |
|--------------------|-----------|----------------------|-----------------------------|
| `--color-bg`       | `#0A0A0A` | rgb(10, 10, 10)      | Page background (OLED Black)|
| `--color-bg-footer`| `#050505` | rgb(5, 5, 5)         | Footer deep black           |
| `--color-red`      | `#FF1A1A` | rgb(255, 26, 26)     | Primary accent, CTAs        |
| `--color-white`    | `#FFFFFF` | rgb(255, 255, 255)   | Headlines, key text         |
| `--color-grey`     | `#888888` | rgb(136, 136, 136)   | Secondary text, labels      |
| `--color-grey-dark`| `#555555` | rgb(85, 85, 85)      | Tertiary text               |

## Glass/Surface Colors

| Token                 | Value                           | Usage               |
|-----------------------|---------------------------------|---------------------|
| `--surface-card`      | `rgba(255,255,255,0.04)`        | Card backgrounds    |
| `--surface-input`     | `rgba(255,255,255,0.06)`        | Input fields        |
| `--border-default`    | `rgba(255,255,255,0.07)`        | Default borders     |
| `--border-red`        | `rgba(255,26,26,0.15)`          | Red accent borders  |
| `--border-red-active` | `rgba(255,26,26,0.35)`          | Hover/active borders|

## Glow Effects

| Usage             | Value                                                    |
|-------------------|----------------------------------------------------------|
| Red button glow   | `box-shadow: 0 0 32px rgba(255,26,26,0.5)`              |
| Card hover glow   | `box-shadow: 0 0 32px rgba(255,26,26,0.08)`             |
| Active input glow | `box-shadow: 0 0 0 3px rgba(255,26,26,0.08)`            |
